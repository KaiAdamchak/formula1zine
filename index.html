About Grid Position

Grid Position is an 8 paged digital and physical zine about 6 unique Formula 1 tracks. Each page of the zine was programmed in p5js and then compiled into a github website. Programming each page allowed for innovative motion and interactive designs with the goal of capturing each track’s special personality. In order to add a personal touch, a series of 50 physical zines were also created to be taken home at the ATLS EXPO Spring 2025.

FAQ

What is a Zine?

Pronounced “zeen”

Noun informal

A zine is usually a short self-published, self-distributed booklet

What is Formula 1 (F1)?

The highest class of international open-wheel single-seater racing that started in 1950

What is P5.js?

A free, open-source JavaScript library designed to make coding for creative applications accessible, especially for artists, designers, educators, and beginners.
